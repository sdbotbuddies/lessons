elif — Syntax

How to write elif in Python 🐍

# Grading system example score = 78 if score >= 90:     print("A — Excellent! 🌟") elif score >= 80:     print("B — Great job! 👍") elif score >= 70:     print("C — Good effort 😊") elif score >= 60:     print("D — Keep trying 💪") else:     print("F — Let's review 📖")

Score is 78 → Python checks each condition top to bottom. 78 < 90, 78 < 80, but 78 >= 70 ✅ → prints "C — Good effort" then STOPS checking.

The Rules
1️⃣

Always starts with if — you must have exactly ONE if at the top.

2️⃣

elif goes in the middle — you can have as many as you need!

3️⃣

else is always LAST — the catch-all when nothing matched.

4️⃣

Python checks top to bottom and stops at the first match!

Structure
if condition1: # checked first
    ...
elif condition2: # only if first failed
    ...
elif condition3: # only if both failed
    ...
else: # nothing matched
    ...

Python Lists

What is a List? 📋

So far each variable holds one thing. A list lets one variable hold many things — in order!

🎒

Real life: A backpack

Your backpack holds many items in pockets. A list is like a backpack — one name, many items inside, each in its own numbered slot.

📓slot 0
✏️slot 1
📐slot 2
🖊️slot 3
🚌

Real life: Seats on a bus

Every seat has a number. Seat 0 is first, seat 1 is next. You can ask "Who is in seat 2?" Python lists work exactly the same — each item has a numbered position.

⚠️ Counting starts at 0, not 1! Slot 0 is the first item.
🐍

In Python

# Create a list with [ ] fruits = ["apple", "mango", "grape"] # Access items by index print(fruits[0]) # apple print(fruits[1]) # mango print(fruits[2]) # grape # How many items? print(len(fruits)) # 3

✏️ Changing items

You can replace any item: fruits[0] = "banana" — like swapping what's in a backpack pocket.

➕ Adding items

fruits.append("kiwi") — like tossing a new item into the backpack at the end.

📏 Length

len(fruits) tells you how many items are in the list — like counting the pockets in your bag.

Loops

Why do we need loops? 🔄

Imagine you have to print 100 student names. Without loops, you'd write 100 lines of code. With a loop — just 3 lines!

😩

Without a loop

print("Hello, Alex")
print("Hello, Beth")
print("Hello, Carlos")
# ...97 more lines 😩
→
🎉

With a loop

for name in students:
    print("Hello, " + name)
# Done! Works for 100 names! 🎉

🔁 for loop

Repeats for each item in a list. You know upfront how many times it will run. Like going through each item on a checklist.

🔃 while loop

Repeats as long as a condition is true. You might not know in advance how many times it'll run. Like waiting until the bus arrives.

for Loop

The for loop — going through a list 📋

Real-life for loops

🎒

Checking a backpack: For each item in your school supplies list, check if you packed it.

🎄

Decorating a tree: For each ornament in the box, hang it on the tree.

📧

Sending invites: For each friend on your list, send them a birthday invite.

🧺

Washing dishes: For each dish in the sink, wash and rinse it.

Key idea: A for loop visits each item in a collection one by one, does something with it, then moves to the next — until the list is finished!

# Print each pizza topping toppings = ["cheese", "pepperoni", "mushroom"] for topping in toppings:     print("Adding:", topping) # Output: # Adding: cheese # Adding: pepperoni # Adding: mushroom
# Count from 1 to 5 using range() for number in range(1, 6):     print("Count:", number) # Output: Count: 1, 2, 3, 4, 5 # range(1, 6) means 1 up to but NOT including 6

while Loop

The while loop — keep going until done! ⏳

Real-life while loops

🚌

Waiting for a bus: While the bus hasn't arrived, keep waiting.

🎮

Playing a game: While you still have lives, keep playing.

🍝

Boiling pasta: While the pasta isn't soft, keep cooking.

💰

Saving money: While savings < $100, keep saving allowance.

⚠️ Warning: If the condition never becomes False, the loop runs forever — this is called an infinite loop! Always make sure something changes inside the loop to eventually stop it.

# Countdown to launch! 🚀 countdown = 5 while countdown > 0:     print(countdown, "...")     countdown = countdown - 1 print("🚀 Blast off!") # Output: 5... 4... 3... 2... 1... Blast off!
# Save up $10 of allowance 💰 savings = 0 while savings < 10:     print("Saved $", savings)     savings = savings + 2 print("🎉 Goal reached! $", savings)

for vs while

Which loop should I use? 🤔

🔁 for loop🔃 while loop
Use when... You know how many times to repeat You repeat until a condition is met
Runs... Once for each item in a list As long as condition is True
Real life Check each item on a grocery list 🛒 Wait until the popcorn stops popping 🍿
Real life Wish happy birthday to each friend 🎂 Keep guessing until you get the right answer 🎯
Real life Stamp each envelope in a pile 📬 Eat until you're full 🍽️
Watch out for... Going out of range (past the list end) Infinite loops! Always update your variable ⚠️
💡

Quick rule: If you have a list → use for. If you're waiting for something to happen → use while.
