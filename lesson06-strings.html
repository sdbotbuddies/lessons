"hello".upper()

"HELLO"

String Methods — Case

Changing text case 🔠

📢
.upper()
ALL CAPS
"hello".upper()
# → "HELLO"

Use for: shouting, usernames, hashtags

🤫
.lower()
all lowercase
"Hello World".lower()
# → "hello world"

Use for: comparing, normalizing text

✍️
.capitalize()
First letter only
"hello world".capitalize()
# → "Hello world"

Use for: names, sentences, titles

🎮 Instagram does .lower() on every username — so "Alex123" and "alex123" are the SAME account!
String Methods — Edit

Cleaning and replacing text ✂️

🧹
.strip()
The Eraser

Removes extra spaces from both ends of a string

⎵⎵⎵hello⎵⎵⎵ .strip() → hello
name = " Alex "
name.strip() # → "Alex"
🌐 Websites use .strip() to clean up form inputs — users accidentally add spaces all the time!
🔄
.replace(old, new)
The Autocorrect

Replaces every occurrence of one thing with another

msg = "I luv pizza"
msg.replace("luv", "love")
# → "I love pizza"
📱 Autocorrect is basically .replace() for your keyboard! It swaps "teh" with "the" automatically.
String Methods — Search & Split

Finding things in strings 🔍

🔍 .find(text)

Returns the index (position) where text first appears. Returns -1 if not found.

"Python rocks".find("rocks")
# → 7
📊 .count(text)

Counts how many times a word or letter appears in a string

"banana".count("a")
# → 3
✂️ .split() — breaks a string into a list!
"Alex Sam Priya"
→ .split() →
"Alex"
"Sam"
"Priya"
names = "Alex Sam Priya".split()  # → ["Alex", "Sam", "Priya"]
📏 len("Python") → 6

len() counts total characters in a string — spaces included! len("Hi there") → 8
