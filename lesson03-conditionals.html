Conditionals

What is an if-else? 🍴

✅

YES Path

The things that happen only when the condition is TRUE.


Example: If it IS raining → grab umbrella

QUESTION
IF / ELSE
SPLIT!
❌

NO Path

The things that happen only when the condition is FALSE.


Example: If NOT raining → leave umbrella home

🛣️

Think of it like a fork in the road. You must pick one path — you can't walk both at the same time. After the fork, both paths lead to the same next town (the next line of code).

Conditional Operations

Comparison operators 🔍

These are just math words you already say in English — now written as symbols!

What you say in real lifeSymbolExample in Python
"You must be at least 48 inches tall">=height >= 48
"The oven is exactly 350 degrees"==temp == 350
"The pizza is NOT done yet"!=done != True
"You have fewer than 3 strikes"<strikes < 3
"Your score is higher than mine">score > my_score
⚡

Key rule: A condition is always either True or False — never "sort of"! Just like asking "Is today Monday?" — it's a yes or a no.

Indentation

The Outline Rule 📐

Essay outline you already know
I. Morning Routine
A. If alarm goes off
→ Get up and brush teeth
→ Eat breakfast
B. Otherwise
→ Keep sleeping
II. Leave for school (always!)
Always runs
Only if alarm
Only if no alarm
# Python uses INDENTATION just like an outline

if alarm_went_off:
    get_up()  # indented = belongs to if
    eat_breakfast()  # indented = belongs to if
else:
    keep_sleeping()  # belongs to else

go_to_school()  # NO indent = always runs!
💡

The golden rule: If a line is indented under an if or else, it ONLY runs when that branch is chosen. No indent = ALWAYS runs, no matter what!

Class Activity

Your Turn! ✏️ The If-Else Challenge

1

Write your if-else in plain English

Think of a decision you make in your own life. Write it as: "If [condition], then [do this]. Otherwise, [do that]."

Example: "If I finish homework before dinner, I get to play video games. Otherwise, I do homework after dinner."

2

Draw the flowchart

Use a diamond for your question and rectangles for the YES and NO actions. Draw arrows connecting them!

3

Label what gets indented

Circle the parts that would be indented in Python (inside the if or else). Put a star ⭐ next to any line that always runs no matter what.

4

Share with the class!

Read your if-else sentence out loud. Can the class guess which symbol goes in the condition — >, <, ==?
