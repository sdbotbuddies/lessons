THE SAME DATA IN PYTHON:

contacts = {
    "Alex":  "555-0101",

    "Priya": "555-0103"

🔑 Each key (name) maps to a value (phone). You look up by name — not by a number!
Dictionaries — Create & Access

Making and using a dictionary 🔑

# Create a dictionary
student = {
    "name" : "Alex",
    "grade": 6,
    "score": 95
}

ANATOMY OF A DICTIONARY:

{ } = dictionary container
"name" = key (always a string)
"Alex" = value (any type!)
: = connects key to value
, = separates pairs

ACCESSING VALUES:

print(student["name"])   # Alex
print(student["grade"]) # 6
print(student["score"]) # 95
📦

Think of keys like labels on a box

You look up by name, not by number!
student["score"] not student[2]

⚠️ KeyError! If you use a key that doesn't exist — Python will crash!
student["age"] → KeyError: 'age'
Dictionaries — Modify

Changing your dictionary 🔧

➕

ADD NEW KEY

contacts["Jordan"]
  = "555-0104"
Like adding a new contact!
{
 "Alex": "555-0101",
 "Sam": "555-0102",
 "Jordan": "555-0104" ✅
}
✏️

UPDATE EXISTING

contacts["Alex"]
  = "555-9999"
Like editing a contact number!
{
 "Alex": "555-9999" ✏️
 "Sam": "555-0102",
 "Priya": "555-0103"
}
🗑️

DELETE

del contacts["Sam"]
Like deleting a contact!
{
 "Alex": "555-0101",
 "Sam": "555-0102" 🗑️
 "Priya": "555-0103"
}
💡 Same syntax for add & update! If the key exists → update it. If it doesn't → create it!
dict["key"] = value works both ways!
Dictionaries — Loops

Going through every item 🔄

THE MAGIC LOOP PATTERN:

for key, value in mydict.items():
key = the label (e.g. "Math")
value = the data (e.g. 88)

STEP BY STEP — WHAT HAPPENS:

Loop 1: subject="Math", score=88 → ✅ Pass
Loop 2: subject="English", score=72 → ✅ Pass
Loop 3: subject="Science", score=55 → ❌ Fail

FULL EXAMPLE — GRADEBOOK:

grades = {
  "Math": 88,
  "English": 72,
  "Science": 55
}

for subject, score in grades.items():
  if score >= 60:
    print(subject, "✅ Pass")
  else:
    print(subject, "❌ Fail")
✨ .items() gives you BOTH the key AND value each loop — use two variables!

Access by Index [0], [1] Key ["name"]
