Functions — Syntax

How to write a function 🐍

def function_name(parameter):
    # indented code goes here
    print("Doing something!")
🎉 Full Example:
def celebrate(name):
    print("🎉 Happy Birthday," + name + "!")
    print("🎂 Hope it's awesome!")

# Calling the function:
celebrate("Priya")
celebrate("Marcus")

Breaking it down 🔍

🔑 def

Short for "define". Tells Python you're creating a function.

📛 function_name

Use lowercase + underscores. Make it describe what it does!

🧪 parameter

The "ingredient" you pass in. Like name in greet(name).

📐 : + indent

Colon starts the body. Everything inside must be indented 4 spaces!

💡 Tip: Just writing def celebrate() doesn't run it! You have to call it by writing celebrate("Alex")

Functions — Return Values

Getting something back with return 📦

def add_tax(price):
    total = price * 1.08
    return total

# Storing the returned value:
my_total = add_tax(10.00)
print("You owe: $", my_total)
# Output: You owe: $ 10.8
🚗 Real Life: The Uber app calls calculate_fare(), it returns your price, and shows it on screen. No return = no price shown!
def double(n):
    return n * 2

result = double(5)
print(result)  # 10
print(double(7)) # 14

print() vs return 📊

Feature print() return
What it does Shows text on screen Sends value back to caller
Analogy Shouting the answer 📢 Handing over a note 📝
Can store? ❌ No ✅ Yes! x = func()

🧠 Remember:

  • print() = show on screen, can't save it
  • return = send back to where it was called
  • You can use both in the same function!

Functions — Parameters

Giving functions ingredients 🧪

0️⃣ No Parameters

Runs the same every time. No customization!

def ring_bell():
  print("🔔 DING!")

ring_bell()
ring_bell()
# Always says DING!

1️⃣ One Parameter

Customize with one piece of info!

def roast(name):
  print(name + " types with
         one finger 🐌")

roast("Alex")
roast("Jordan")

2️⃣ Multiple Params

Pass in multiple ingredients!

def add(a, b):
  return a + b

print(add(3, 5))
# 8
print(add(10, 20))
# 30
🎮 Think of it like a video game: spawn_enemy('dragon', 100) — the function needs name AND health points. Two parameters! The dragon spawns with exactly those values!
